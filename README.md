# docs.MagicAF

[![Deploy Hugo site to Pages](https://github.com/intracav/docs.MagicAF/actions/workflows/hugo.yml/badge.svg)](https://github.com/intracav/docs.MagicAF/actions/workflows/hugo.yml)

Documentation site for MagicAF - Defense-grade, HIPAA-compliant AI toolkit for secure, air-gapped deployments.

## About

This repository contains the Hugo-based documentation website for [MagicAF](https://github.com/magicaf/magicaf), a production-ready Rust framework for RAG pipelines, NLP analysis, embeddings, vector search, and LLM orchestration. Designed for SIPR/NIPR, classified, and healthcare environments with zero cloud dependencies.

## Live Site

Visit the documentation at: https://magicaf.intracav.ai/

## Local Development

To run the documentation site locally:

```bash
# Build the site
make build

# Serve locally (if available)
hugo server
```
